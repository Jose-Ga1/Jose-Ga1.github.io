get.gbif <- function(genus, species, na.rm = TRUE, crop = TRUE) {
    # GBIF query
    gps <- gbif(genus, species, geo = FALSE)
    # drop NA entries; note that this option is visited only with na.rm=TRUE
    if (na.rm) {
        gps <- subset(gps, !is.na(lon) & !is.na(lat))
    }
    if (crop) {
        keep <- c(which(colnames(gps) == "species"), which(colnames(gps) == 
      "lat"), which(colnames(gps) == "lon"), which(colnames(gps) ==
            "month"))
        gps <- gps[keep]
    }
    return(gps)
}

Using reproductive isolation using this method can be problematic because it doesn’t account for any factor in species delination other than time. The matter stands that even though there may no overlap in flowering time but the pollen may be resistant to dessication and could fertilize another plant during a different season leading to a hybrid. Additionally, just because species may have similar or identical flowering periods isn’t entirely an indicator that the species are the same or related, it may just be more favorable to reproduce at specific.

Setosa is very morphologically distinct from Versicolor and Virginica evident by having smaller mean petal and sepal width/lengths. On the contrary Virginica and Versicolor are more closely related but still have distinguishable morphologies. Geographically, Setosa has no overlap with the other 2 flowers, but there is overlap between Virginica and Versicolor, including regions for possible hybridization events to occur. The temporal data suggests that there is overlap in flowering periods between the 3 flowers weakening the argument that temporal factors was a sole factor in speciation. All 3 species have the same chromosome count, so it can be aasumed that chromosomal differences are not the driving factor in reproductive isolation. The most compelling piece of evidence is the morphological differences which show clear differences among the 3 iris species (especially Setosa), and is supported by LDA. That being said Anderson’s original species designation seemed largley correct, and the data added after the hypothesis appeared to strenghten the original assertion. Based on the evidence collected it is observed that there is 3 iris species, due to the lack of chromosomal differences, the distinct morphology, and the geographical seperationn.
